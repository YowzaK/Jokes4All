Instructions for running the web application (jokes4all);

Navigate to jokes4all\src\configs, then open the constants.ts file and edit the API endpoints to what they are runnin in your machine.

Open CMD prompt and type the below code to run the project;

- npm install (wait for it to finish installing)

- ng serve (to run the webapp on locahost)

After the above commands the webapp should be up and running. 

Use the .env files provided with the API's to change port numbers, other endpoints and SQL configuration.
